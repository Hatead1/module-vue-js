# Vue.js Rete.js Modules

## Project setup

### Clone repository
```
git clone https://github.com/Hatead1/module-vue-js
```
### Change directory
```
cd module-vue-js/
```
### Install npm modules
```
npm i
```

### Compiles and hot-reloads for development
```
npm run serve
```
### Open in browser
```
http://localhost:8080/
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
